# Playto-Community
A full-stack prototype built using React and Django REST Framework that demonstrates a scalable community feed with nested discussions, safe interaction handling, and a dynamic leaderboard system.  This project focuses on performance, data integrity, and clean architecture, following real-world backend and frontend design practices.
# Clone the repository
git clone <your-repo-url>
cd backend

# Create virtual environment
python -m venv venv

# Run database migrations
python manage.py migrate

# Start the development server
python manage.py runserver

cd frontend

# Install dependencies
npm install

# Start React app
npm start
